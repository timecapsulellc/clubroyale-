# App Icon Design Specifications

Since image generation was rate-limited, use these specifications to create icons:

---

## App Icon (512x512)

### Design Concept
Modern card game icon with playing cards against gradient background

### Colors
- **Primary Gradient:** Deep Purple (#7C3AED) to Cyan (#06B6D4)
- **Cards:** White with vibrant suit colors
- **Accents:** Gold (#FFD700) for premium feel

### Elements
- 3-4 playing cards in fan arrangement
- Visible suits: Hearts (red), Spades (black), Diamonds (red)
- Cards should have rounded corners
- Subtle shadow for depth
- NO text on icon

### Style
- Flat design with subtle 3D depth
- Clean, minimal
- Suitable for small sizes (48px and up)
- Vibrant but not overly busy

---

## Feature Graphic (1024x500)

### Design Concept
Wide banner showcasing TaasClub brand and games

### Layout
- Left side: TaasClub logo/text
- Center: Floating playing cards with dynamic arrangement
- Right side: "Play with Friends" tagline
- Background: Same purple-to-cyan gradient

### Text Elements
- "TaasClub" - Large, modern gaming font
- "Marriage & Call Break" - Subtitle
- "Play with Friends Online" - Tagline

### Card Suits
- Scatter suit symbols (♠ ♥ ♦ ♣) decoratively
- 3D floating card effect

---

## Adaptive Icon (Android)

### Foreground Layer
- Playing cards only (no background)
- Centered, with safe zone margins

### Background Layer
- Solid deep purple (#7C3AED)
- OR gradient texture

---

## Tools to Create

### Free Options
- **Canva** - Templates for app icons
- **Figma** - Professional design tool (free tier)
- **GIMP** - Open source image editor

### AI Image Generators
- **Midjourney** - High quality card game icons
- **DALL-E 3** - Detailed icon generation
- **Leonardo.ai** - Game art focus

### Prompt for AI Generators
```
A modern mobile app icon for a card game called TaasClub. 
Features playing cards (hearts, spades, diamonds) in a dynamic 
fan arrangement against a gradient background from deep purple 
to cyan. Clean, minimal design with subtle 3D depth and glossy 
finish. Professional mobile gaming aesthetic. No text. 
512x512 pixels.
```

## Screenshot Specifications

### Required Sizes
- **Phone:** 1080x1920 pixels (9:16)
- **Tablet:** 1200x1920 pixels (7" tablet)

### Recommended Screenshots (4-8)
1. Home screen / Lobby
2. Creating a room
3. Active gameplay (Marriage)
4. Active gameplay (Call Break)
5. Score/Settlement screen
6. Diamond store
7. Invite friends feature

### Screenshot Style

- Highlight key features
- Add brief captions
- Use consistent branding colors
